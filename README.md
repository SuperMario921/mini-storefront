# Mini-Storefront

A mini-storefront example project using Next.js and React. Browse products, filter by price, add items to the cart, and see a live cart summary.

## Setup
```bash
git clone https://github.com/username/my-shop-app.git
npm install
npm run dev
Open http://localhost:3000

Features
-Product listing & stock info
-Price filter
-Add/remove items from cart
-Cart summary & total
-Loading/error/empty messages
